    Transfer code for FCMB

    If you are a First City Monument Bank customer, you have probably wondered how to make the transaction process as simple and easy as possible. Fortunately, the bank provides such an option to its customers, and they can easily perform various procedures using their mobile devices.

    Any transaction exceeding N500,000 per day requires a transaction code. To generate a transaction code, adhere to the steps listed below.

  • Dial *329*0#.
  • Select create/reset code.
  • Select your ATM card (for customers with multiple cards).
  • Enter your ATM card PIN.
  • Create a 4-digit transaction code, e.g. 1234.
  • Enter your USSD PIN.
  • Your transaction code will be successfully generated.

  • Next, select "Dispute Resolution" and then proceed to choose your account.
